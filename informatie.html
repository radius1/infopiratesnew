InfoPirates , een project van klas MED2A

InfoPirates is een 'quest based' QR code game, gemaakt om bezoekers tijdens de open dag extra interactiviteit mee te geven aan hun bezoek.

met behulp van een mobiele telefoon kunnen bezoekers 'quests' volbrengen om op deze manier kans te maken op leuke prijzen.

Deze mobiele game is gemaakt door middel van HTML, CSS, Javascript en PHP, De voornaamste talen op de opleiding Mediadevelopment. Met behulp van de populaire GIT cloud-dienst github hebben de studenten in projectvorm samengewerkt tot dit mooie resultaat.

Als docent van deze enthousiaste groep developers kan ik ook niks anders dan ontzettend trots zijn hoe zij dit project hebben volbracht.

Deze game is een voorbeeld van de projecten die worden uitgevoerd op de opleiding Mediadevelopment op het Radius College. Lijkt dit je interessant? neem dan een kijkje op www.radiuscollege.nl en zoek ons op!
